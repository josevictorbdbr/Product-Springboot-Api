# Product-Springboot-Api

API REST basica para gerenciar produtos, o projeto permite o gerenciamento de produtos, incluindo criação, listagem, atualização e exclusão.

Tecnologias utilizadas:

- Java 17
- Spring Boot
- Spring Data JPA
- Maven

Funcionalidades:

- Criar produtos
- Listar produtos ativos
- Atualizar informaçoes(exceto id) de um produto
- Buscar produtos pelo id
- Remover produto usando id para desativalo
